What is SciCrunch

SciCrunch is a data sharing and display platform. Anyone can create a custom portal where they can select searchable subsets of hundreds of data sources, brand their web pages and create their community. SciCrunch will push data updates automatically to all portals on a weekly basis. User communities can also add their own data to scicrunch, however this is not currently a free service.

Select a Component to Add

Parralax Slider

A slider with a single background image that shifts a little with each slide. New slides consist of a title, description, and small image that links elsewhere.

Parralax Counter

An unchangable counter that you can set the image for. The stats shown are dynamic stats related to your community.

Search Banner

A search bar above a background image you upload so direct users to the proper place in Community Resources.

Image Slider

A traditional slider where each slide has a different background image. Each slide has a title, description, and background image to link to elsewhere.

Dark Parralax Counter

An unchangable counter that you can set the image for that is better for darker backgrounds. The stats shown are dynamic stats related to your community.

Goto Block

A component to provide a clear button for a user to go to. Useful for directing to join the community or find something specific.

News Thumbnails

Displays 8 articles that you have to add separately to the component. Useful for showing off recent news or events.

Calendar Block

A simple block that is half calendar, half youtube video.

Search Bar

A small search bar that can be placed anywhere for searching through Community Resources

Text Block

Simple text block that you can upload an image next to and set the background color.

Video Text

Another simple text block that has an image next to it instead.

RSS Feed

Half text block next to an RSS feed.

Categories Block

A dynamic category section with a small twitter feed to the right. Categories have to be added separately to the component and provide an icon.

Services Block

3 link boxes where you control the text, icons, and link for each box.

White Services Block

3 static boxes that show just the text.

Works Thumbnails

4 thumbnails that you add separately with an image, text, and a link.

Page Box

Full page text, preferably short and simple.

Dynamic Services

A 4 block text where you control the color and add the blocks dynamically to the component.

Image Search Box

A simple search bar with an image you provide above it.

Sponsors Box

A scrolling list of sponsors (or any other list of items).
